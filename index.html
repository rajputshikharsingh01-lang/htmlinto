HTML

Introduction:

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It tells the browser how to display text, images, links, videos, and other elements on a webpage.

Key Points:

  • Markup Language → It uses "tags" (like <p>, <h1>, <a>) to define structure.
  • Not a Programming Language → It doesn’t perform logic or calculations, but organizes content.
  • Works with CSS & JavaScript →
    • HTML → Structure
    • CSS → Styling (colors, layout, fonts)
    • JavaScript → Interactivity
